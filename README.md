DoneWithIt is a marketplace app that allows users to sell items they no longer need and find great deals on second-hand items. Built with [Expo](https://expo.dev), it uses modern technologies to create a seamless cross-platform experience for iOS, Android, and web.

---

## Prerequisites

Before starting, ensure you have the following installed:

- **Node.js**: v20+  
- **npm**: v10+  
- A code editor (like [VS Code](https://code.visualstudio.com/))  
- [Expo CLI](https://docs.expo.dev/get-started/installation/) (installed globally via `npm install -g expo-cli`)
